The Context

Mobile Phone Usage

owned a cell phone.

GPS Usage

Previous researchers have used GPS to track participant movement. However, GPS cannot track movement indoors.

Friendship

Social support in college leads to increased mental health, decreased stress levels, and overall wellbeing.

WiFi Triangualtion

WiFi can be used to track participant location, but new apps and testing need to be done.

Project Overview

Participants were asked to use Android smartphones preinstalled with our software that could record and send phone usage data and colocation information to our server located in the university. The software application ran on any android mobile phone, and would mandatorily start at bootup. We used the WiFi fingerprinting method for indoor localization, which achieved an accuracy of 5-7 meters in our study. We also implemented periodic Bluetooth scanning to discover participants that were close to each other. This study followed participants for one academic year.

Participants were also asked to complete self-reported questionnaires on language use, friendship tie-strength, and adaptation to college life, once each term (~every 13 weeks).
